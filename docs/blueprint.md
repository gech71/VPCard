# **App Name**: Prepaid Portal

## Core Features:

- Balance Inquiry: Fetch and display the current balance for a specified card.
- Transaction History: Retrieve and list the last N transactions for a card.
- POS Limit Management: Set and update the Point of Sale (POS) limit for a card.
- ATM Limit Management: Set and update the Automated Teller Machine (ATM) limit for a card.
- PIN Change: Enable a secure PIN change flow for a card.
- Card Information Display: Display card details such as status, masked card number, expiry date and card type.

## Style Guidelines:

- Primary color: Deep blue (#0F4C81) to convey trust and security, in alignment with financial services.
- Background color: Light blue (#D1E5F5), a desaturated shade of the primary, providing a clean and unobtrusive backdrop.
- Accent color: Teal (#368B85), offering a contrasting, yet harmonious color for interactive elements.
- Body and headline font: 'Inter', a sans-serif font known for its modern and readable design, will ensure clarity across all text elements.
- Use a clean and responsive layout with clear sections for each feature. Ensure that all elements are easily accessible on various screen sizes.
- Incorporate minimalist icons to represent different actions and card information categories, enhancing usability and visual appeal.
- Implement subtle transitions and loading animations to enhance user experience, providing feedback during API calls and page transitions.
